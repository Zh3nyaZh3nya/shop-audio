# Разработка веб-приложения для магазина аудиотехники

Этот проект является дипломной работой для **Caspian College** на тему:
> "Разработка сайта интернет-магазина по продаже электронной техники."

## Стек технологий

- **Frontend**: [Nuxt 3](https://nuxt.com/docs/getting-started/introduction) — современный фреймворк для создания Vue-приложений.

## Основные возможности

- **Каталог аудиотехник**: Просмотр ассортимента товаров с фильтрацией по категориям, брендам, ценам и другим параметрам..
- **Корзина**: Удобное добавление товаров в корзину.
- **Адаптивный дизайн**: Приложение корректно отображается на всех устройствах — от смартфонов до настольных компьютеров.


### Требования
